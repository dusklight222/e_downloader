# e_downloader

鹅小通付费可观看视频下载

## 流程

### 服务端

1. 配置文件填写
   - 数据库地址
   - 邮件地址(用于发送给本人)
2. 配置tcp交互服务端
   - 每次收到来自客户端的邮件地址，数据库表中对应数据-1，回复OK



### 客户端

1. 配置文件填写
   - 下载模式(单URL还是多URL)
   - 文件位置填写(前提是多URL)
   - 邮件地址填写
2. 配置tcp交互客户端
   - 每次下载前，都要向服务器发送邮件地址
   - 成功接收OK，方可进行下一步操作
3. 下载URL
   - 内嵌m3u8下载器
   - 对URL进行调整，变为m3u8格式
   - 下载
